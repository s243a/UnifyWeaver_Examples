# UnifyWeaver Examples

Optional, larger, or generated examples for UnifyWeaver.

This repository is intended to be mounted from the main UnifyWeaver repository
as the `examples/more` submodule. Keep generated benchmark artifacts and
non-essential example material here so the main repository stays focused on
source, tests, and small runnable examples.

## Layout

- `graph/effect_dist/haskell/gen/` - generated Haskell effective-distance
  benchmark material.
